Menü

}

Randspalte

Diese Spalte besitzt eine feste Breite und wird durch die Eigenschaft float:right automatisch am rechten Rand platziert.

Mittlere Spalte

div#main {
  width: auto;
  padding: 10px;
  border: 2px #aaa solid
  margin: 0 224px 0 224px;
  ...
}

Die Breite für den linken und rechten Margin ergibt sich entsprechend des Box-Modells aus den Eigenschaften width, padding und border zu: 200 px + 2 * 10 px + 2 * 2 px = 224 Pixeln

Fußzeile

  clear: both;

}

Die Fußzeile (#footer) steht immer unterhalb der Spalten. Dafür sorgt die Eigenschaft: clear:both.

Der Test-Hotfix bestand einfach in einer Erweiterung des Inhaltes der Fußzeile.
